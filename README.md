## Install

```sh
yarn add @micro-app/vue-cli
```

or

```sh
npm install -S @micro-app/vue-cli
```

## Usage

可通过配置 `process.env.VUE_CLI_SERVICE_CONFIG_PATH` 设置配置文件路径
